  We've just opened an experimental nomadbase in Lisbon. We hope to run it till end of January 2016.

  We're looking forward to creating this place with you! So let us know and welcome. :)

  By default we're welcoming long term travelers, nomads, hitchhikers etc. And if you don't belong to these groups feel free to write to us and we'll see if we have space for you.

  In the spirit of nomadbase, the staying is free.

  More about the concept is explained on nomadwiki, nomadbase.info, nomadbase.org.
